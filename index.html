Quick Installation Guide

1

Include CSS & JS

Add the widget files to your HTML document:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/kk-whatsapp-widget/dist/style.css"> <script src="https://cdn.jsdelivr.net/npm/kk-whatsapp-widget@latest/dist/index.umd.js"></script>
2

Add Container Element

Create a div where the widget will be rendered:

<div id="whatsapp-widget"></div>
3

Initialize Widget

Use the generated code from above to initialize your widget:

window.whatsappWidget("#whatsapp-widget", { // Your configuration here });
