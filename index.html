WhatsApp Widget

Interactive Documentation & Live Demo

Key Features

Multi-Member Support

Display multiple team members with individual WhatsApp contacts and avatars.

Responsive Design

Works seamlessly on desktop and mobile devices with touch-friendly interactions.

Customizable

Fully customizable styling with CSS classes and configuration options.

Lightweight

Built with vanilla TypeScript, no heavy dependencies or frameworks required.

API Reference

Configuration Interface

interface WidgetConfig {
  styles?: {
    button?: {
      backgroundColor?: string;
      color?: string;
    };
  };
  members: {
    name: string;      // Team member's display name
    phone: string;     // WhatsApp phone number with country code  
    avatar: string;    // URL to member's profile image
  }[];
}

Parameters

Parameter Type Required Description
selector string Yes CSS selector for the container element
config WidgetConfig Yes Configuration object with members and optional styles
config.members Member[] Yes Array of team members with name, phone, and avatar
config.styles StyleConfig No Optional styling configuration for the widget button

Integration Guide

1. Include Required Dependencies

<!-- Font Awesome for icons -->
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0-beta3/css/all.min.css" />

<!-- Widget Styles -->
<link rel="stylesheet" href="path/to/widget-styles.css" />

2. Add Widget Container

<!-- Widget will be inserted here -->
<div id="whatsapp-widget"></div>

3. Initialize Widget

<script src="path/to/widget.js"></script>
<script>
  document.addEventListener("DOMContentLoaded", () => {
    window.whatsappWidget("#whatsapp-widget", {
      members: [
        {
          name: "Your Name",
          phone: "+1234567890",
          avatar: "https://your-avatar-url.jpg"
        }
      ]
    });
  });
</script>

CSS Customization

Available CSS Classes

CSS Class Description
.kk_wa-button Main WhatsApp floating button
.kk_wa-chat-container Chat popup container
.kk_wa-chat-header Header section of the chat popup
.kk_wa-member-card Individual member card
.kk_wa-member-avatar Member avatar container
.kk_wa-member-info Member name and phone information
