üì¶ Installation Methods


              
              

                

                  üåê CDN Installation (Recommended)
                

                

                  The easiest way to get started. Just add these two lines to
                  your HTML:
                


                

                  
<!-- Add to your <head> section -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/kk-whatsapp-widget/dist/style.css">

<!-- Add before closing </body> tag -->
<script src="https://cdn.jsdelivr.net/npm/kk-whatsapp-widget@latest/dist/index.umd.js"></script>

                

              


              
              

                

                  üì¶ NPM Installation
                

                

                  For projects using a build system like webpack, Vite, or
                  Parcel:
                


                

                  
# Using npm
npm install kk-whatsapp-widget

# Using pnpm (recommended)
pnpm add kk-whatsapp-widget

# Using yarn
yarn add kk-whatsapp-widget

                


                

                  Then import in your JavaScript:
                

                

                  
import 'kk-whatsapp-widget/dist/style.css';
import { whatsappWidget } from 'kk-whatsapp-widget';

                  üîß Framework Integration Examples
                


                
                

                  
React

                  

                    
import React, { useEffect, useRef } from 'react';
import 'kk-whatsapp-widget/dist/style.css';

const WhatsAppWidget = () => {
  const widgetRef = useRef(null);

  useEffect(() => {
    if (widgetRef.current && window.whatsappWidget) {
      window.whatsappWidget(widgetRef.current, {
        members: [
          {
            name: "Support",
            phone: "+1234567890",
            avatar: "https://placehold.co/400x400?text=Support",
            isOnline: true,
          },
        ],
      });
    }
  }, []);

  return <div ref={widgetRef}></div>;
};

                  üîß Troubleshooting
                


                

                  

                    
Widget not showing?

                    

                      Make sure you've included both the CSS and JavaScript
                      files, and that the DOM element exists when you initialize
                      the widget.
                    

                  


                  

                    
Console errors?

                    

                      Check that all required properties are provided in your
                      configuration object, especially the 'members' array.
                    

                  


                  

                    
Styling issues?

                    

                      Ensure the CSS file is loaded before the widget
                      initializes, and check for CSS conflicts with your site's
                      styles.
                    

                  


                  

                    
Mobile problems?

                    

                      The widget is responsive by default. Check your viewport
                      meta tag and ensure you're not overriding the widget's
                      mobile styles.
